1.7 ЗМІНА РОЗМІРУ ЗОБРАЖЕННЯ

 

Для зміни розміру зображення тільки на екрані розглянемо декілька кнопок панелі управління системи КОМПАС, які використовуються для цих цілей (рис.7):

«Увеличить масштаб рамкой» - кнопка для збільшення частини екрану до розмірів екрану рамкою, заданою двома точками по діагоналі. Після виклику команди  зовнішній вигляд курсора зміниться: він перетвориться на перехрестя. Вкажіть перший кут, потім переміщайте курсор для досягнення потрібного розміру рамки. На екрані відображатиметься фантом рамки. Вкажіть другий кут рамки. Масштаб зображення збільшиться так, щоб область, обмежена рамкою, повністю уміщалася у вікні документу.

«Увеличить масштаб» - кнопка дозволяє збільшити масштаб  зображення в активному вікні в певну кількість разів, встановлену за умовчанням в налаштуваннях системи.

«Уменьшить масштаб» - кнопка для зменшення масштабу зображення на екрані.

«Сдвинуть» - кнопка для переміщення електронного креслення по екрану при  переміщенні курсора по екрану. Після виклику команди курсор міняє свою форму на чотиристоронню стрілку.

«Приблизить/отдалить» - кнопка для збільшення або зменшення зображення на екрані, що дозволяє плавно міняти масштаб, наближаючи або віддаляючи зображення. Для цього натисніть ліву кнопку миші і, не відпускаючи її, переміщайте курсор у вертикальному напрямі. При русі курсора вгору зображення плавно збільшуватиметься, у зворотному напрямі - зменшуватися. Центром панорамування є точка, в якій була натиснута ліва кнопка миші. Якщо Ви користуєтеся мишею з колесом, то для панорамування зображення обертайте колесо миші.

«Обновить изображение» - кнопка для перемальовування креслення і видалення "сміття" з поля креслення.

«Показать все» - кнопка для виведення на екран усього креслення або усіх зображень фрагмента.
